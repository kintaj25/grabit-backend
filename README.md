# Grab It - Booking Service\n\nReal-time seat booking service with distributed locking for handling high concurrency.\n\n## Features\n- Real-time seat locking using Redis\n- Distributed transaction support\n- Seat reservation with automatic TTL expiration\n- Booking confirmation and cancellation\n- Concurrent request handling without double-booking\n\n## Architecture\n- Node.js + Express + TypeScript\n- PostgreSQL for persistent storage\n- Redis for distributed seat locks\n- TTL-based automatic lock expiration (5 minutes)\n\n## Setup\n1. Create .env file from .env.example\n2. Install dependencies: npm install\n3. Run: npm run dev\n\n## API Endpoints\n\n### Booking Operations\n- POST /api/bookings - Create booking\n- PATCH /api/bookings/:bookingId/confirm - Confirm booking\n- PATCH /api/bookings/:bookingId/cancel - Cancel booking\n- GET /api/bookings/:bookingId - Get booking\n- GET /api/bookings/user/:userId - Get user bookings\n\n### Seat Operations\n- POST /api/seats/lock - Lock seats\n- POST /api/seats/unlock - Unlock seats\n- GET /api/seats/status/:showId/:seatId - Get seat status\n- GET /api/seats/show/:showId - Get all seats status for show\n
